## üìä Estrategia de Puntuaci√≥n

### Estrategia Actual: All-or-Nothing

- ‚úÖ **Todos los tests pasan:** `pointsEarned = experiencePoints` (50/50)
- ‚ùå **Alg√∫n test falla:** `pointsEarned = 0` (0/50)

**C√≥digo:** [`SolutionCommandServiceImpl.java:255-275`](../src/main/java/com/levelupjourney/microservicechallenges/solutions/application/internal/commandservices/SolutionCommandServiceImpl.java#L255-L275)

```java

    return maxPoints;
}
return 0; // No points if not all tests pass
```

### Estrategia Alternativa: Proporcional

Para habilitar puntos proporcionales:

```java
if (allPassed) {
    return maxPoints;
}
// Uncomment for proportional scoring:
return (maxPoints * passedTests) / totalTests;
```

**Ejemplo:**
- 2 de 4 tests pasan ‚Üí `pointsEarned = 25` (50 * 2 / 4)
